# Violentmonkey Dark Theme
A dark theme for Violentmonkey using the [Dracula](https://github.com/codemirror/CodeMirror/blob/master/theme/dracula.css) editor theme.

## Installing
Copy and paste the full content of [theme.css](https://raw.githubusercontent.com/synthtech/violentmonkey-theme/master/theme.css) to Settings -> Advanced -> Custom Style and click save.
